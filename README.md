OpenDoor – AI Real Estate CRM with Voice Assistant

OpenDoor is an AI-powered Real Estate CRM platform that helps agencies manage leads, simulate sales calls, and automate client communication using a voice-enabled AI assistant. It combines lead scoring, sentiment analysis, and CRM analytics in a modern dashboard to improve sales performance.

✨ Features

📊 Lead Management Dashboard

🤖 AI Voice Call Simulator

😊 Sentiment Analysis of Conversations

🎯 Lead Scoring & Readiness Prediction

📧 Automated Client Communication

📅 Appointment & Pipeline Tracking

📈 CRM Analytics & Metrics

🔐 Authentication & User Sessions

🧠 AI Capabilities

Voice-based sales conversation simulation

Real-time sentiment detection

Lead qualification scoring

AI-assisted CRM insights

🚀 Installation
# Clone repo
git clone https://github.com/yourusername/OpenDoor.git

# Frontend
cd OpenDoor
npm install
npm run dev

# Backend
cd backend
npm install
npm start
⚙️ Environment Variables

Create .env file:

VITE_SUPABASE_URL=your_url
VITE_SUPABASE_ANON_KEY=your_key
GROQ_API_KEY=your_key
